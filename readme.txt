This plugin allows you to use a shortcode anywhere to display comments on WordPress pages and posts along with the comment form.

== Description ==

This plugin allows you to display comments and comment form anywhere in WordPress. The comments list can be displayed for WordPress pages and posts without modifying the code and it works with most of the available themes. Just install and activate this plugin and use [sp_comments_block] shortcode anywhere in WordPress to display comments list and comment form.<br><br>Read more about this plugin on <a href="https://siriuspro.pl">WordPress</a> blog.

== Installation ==

1. Unzip the comments-shortcode.zip folder.
2. Upload the comments-shortcode folder to your /wp-content/plugins directory.
3. In your WordPress dashboard, head over to the Plugins section.
4. Activate Comments Shortcode plugin.
